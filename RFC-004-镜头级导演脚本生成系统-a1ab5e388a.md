## 验收标准

### Scene → Shot 拆分模块

- 系统可以自动将Scene拆分为Shot，每个Shot满足单一视觉焦点、单一主要动作或情绪，可在2–6秒内表达清楚
- 拆分策略包括动作变化、情绪变化、视角变化和对白切换

### 镜头属性生成模块

- 系统可以自动生成摄影属性（shotSize、cameraMove、angle）
- 系统可以自动生成构图属性（角色站位、视线方向、前景 / 背景层次、构图比例）
- 系统可以自动生成情绪与动作（角色表情、氛围关键词、关键动作描述）
- 系统可以自动生成单镜头时长（1.5s – 8s，可配置），基于镜头类型、对白长度和StyleProfile节奏
- 系统可以自动生成AI图像生成提示词（imagePrompt），用于后续的画面生成

### 台词与旁白绑定

- 系统可以自动绑定台词与旁白，每句对白必须绑定一个Shot，可选展示方式（屏幕字幕、旁白、角色内心独白）

### ShotList 生成与管理

- 系统可以自动生成结构化的ShotList，包含shotId、sceneId、order、characters、camera、composition、action、emotion、dialogueRef、durationSec和flags
- 系统可以管理ShotList的版本状态，包括草稿、人工调整、下游使用和废弃

### 导演备注

- 用户可以输入导演备注，系统保存备注信息

### 关键校验规则

- 系统可以自动校验ShotList，包括Σ durationSec ≈ 单集时长、所有对白均被绑定、相邻镜头角色连续性合理和镜头密度符合StyleProfile
- 系统可以根据校验结果，阻断不符合要求的ShotList进入分镜图生成，ShotList未锁定禁止进入分镜图生成，对白未绑定红色阻断，时长严重超标强制重生成

### 人工介入设计

- 用户可以合并或拆分镜头，系统保存修改后的ShotList
- 用户可以手动修改镜头时长，系统保存修改后的时长
- 用户可以标记关键镜头（封面 / 高潮），系统保存标记状态
- 用户可以锁定镜头防止AI覆盖，系统保存锁定状态

## API接口

### 生成ShotList

```http
POST /api/v1/projects/{id}/scripts/{version}/shot-lists
Content-Type: application/json

{
  "characterAssetsId": "character-assets-id"
}
```

### 获取ShotList列表

```http
GET /api/v1/projects/{id}/scripts/{version}/shot-lists
```

### 获取ShotList详情

```http
GET /api/v1/projects/{id}/scripts/{version}/shot-lists/{version}
```

### 更新ShotList

```http
PUT /api/v1/projects/{id}/scripts/{version}/shot-lists/{version}
Content-Type: application/json

{
  "shots": [
    {
      "shotId": "shot_08",
      "sceneId": "scene_03",
      "order": 1,
      "characters": ["char_01", "char_02"],
      "camera": {
        "shotSize": "medium",
        "move": "static",
        "angle": "eye_level"
      },
      "composition": {
        "layout": "two_shot",
        "focus": "char_01"
      },
      "action": "主角回头",
      "emotion": "紧张",
      "dialogueRef": ["dlg_12"],
      "durationSec": 3.5,
      "imagePrompt": "A young man in school uniform turns back with a tense expression, medium shot, eye level angle, detailed anime style, vibrant colors",
      "flags": ["key_shot"]
    }
  ]
}
```

### 锁定ShotList

```http
POST /api/v1/projects/{id}/scripts/{version}/shot-lists/{version}/lock
```

### 解锁ShotList

```http
POST /api/v1/projects/{id}/scripts/{version}/shot-lists/{version}/unlock
```
