# AI漫剧生成平台

本项目是一个AI驱动的平台，旨在将小说文本转化为漫剧风格的视频。它提供了一个完整的流水线，引导用户通过结构化的四个步骤，最终生成一份可用于制作漫剧视频的、完整的导演脚本。

## 功能特性

平台遵循严格的、顺序化的四步工作流：

1.  **项目与风格设定：** 定义全局项目配置、视觉美学、叙事节奏和AI行为约束。
2.  **小说到剧本生成：** 将原始小说文本转换为结构化剧本，识别场景、对话、旁白和角色情绪。
3.  **角色设计与资产库：** 从剧本中提取角色，生成详细的角色设定卡，并为每个角色创建一致的视觉资产。
4.  **镜头级导演脚本生成：** 将结构化剧本分解为独立的镜头，包含机位、构图、动作、对白绑定和预估时长。

## 技术栈

-   **前端:** React (使用 Vite), Tailwind CSS
-   **后端:** Node.js (使用 Express)
-   **数据库 (模拟):** 用于演示的内存数据存储

## 快速上手

请遵循以下说明在您的本地机器上设置并运行本项目。

### 环境要求

-   [Node.js](https://nodejs.org/) (推荐 v18 或更高版本)
-   [npm](https://www.npmjs.com/) (通常随 Node.js 一起安装)

### 安装步骤

1.  **克隆仓库:**
    ```bash
    git clone <repository_url>
    cd AI-Novel-Movie
    ```
    (注意: 如果这是一个真实仓库，请将 `<repository_url>` 替换为实际的URL)

2.  **后端设置:**
    进入 `server` 目录并安装依赖：
    ```bash
    cd server
    npm install
    ```

3.  **前端设置:**
    进入 `client` 目录并安装依赖：
    ```bash
    cd client
    npm install
    ```

### 运行应用

请确保前端和后端服务同时运行。

1.  **启动后端服务:**
    在 `server` 目录下运行:
    ```bash
    npm start
    ```
    后端API将在 `http://localhost:3000` 上可用。

2.  **启动前端应用:**
    在 `client` 目录下运行:
    ```bash
    npm run dev
    ```
    前端开发服务器通常会在您的浏览器中打开 `http://localhost:5173`。

## 使用方法

本应用将引导您顺序完成四个步骤。目前，您可以通过在 `client/src/App.tsx` 文件中取消注释相应的组件来切换和查看不同步骤的UI。

### 第一步：项目与风格设定
定义您项目的基础信息和整体风格。

### 第二步：小说剧本生成
粘贴您的小说文本以生成结构化剧本。

### 第三步：角色设计与资产库
提取和定义角色细节，并生成视觉资产。

### 第四步：镜头级导演脚本生成
为视频制作生成最终的、详细的镜头列表。
