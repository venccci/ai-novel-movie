# AI漫剧生成平台

本项目是一个AI驱动的平台，旨在将小说文本转化为漫剧风格的视频。它提供了一个完整的流水线，引导用户通过结构化的四个步骤，最终生成一份可用于制作漫剧视频的、完整的导演脚本。

## 功能特性

平台遵循严格的、顺序化的四步工作流：

1.  **项目与风格设定：** 定义项目名称、视觉风格（日式动漫、3D动画、赛博朋克、黑白素描）、画幅比例，以及自定义正向/负向提示词。
2.  **小说到剧本生成：** 将原始小说文本转换为结构化剧本，识别场景、地点、时间、角色和描述，支持手动编辑和添加场次。
3.  **角色设计与资产库：** 从剧本中提取角色，生成角色设定（名称、角色、特征），支持AI生成角色立绘和手动调整。
4.  **镜头级导演脚本生成：** 将剧本转化为分镜脚本，包含画面描述、镜头类型、运镜方式、音频和标签，支持逐个镜头生成预览图。

## 技术栈

-   **前端:** React (使用 Vite), Tailwind CSS, TypeScript
-   **AI 集成:** DeepSeek API (用于剧本分析、角色提取、分镜生成)
-   **后端 (可选):** Node.js (使用 Express) - 用于扩展存储和项目管理功能
-   **状态管理:** React Hooks (useState, useEffect)

## 快速上手

请遵循以下说明在您的本地机器上设置并运行本项目。

### 环境要求

-   [Node.js](https://nodejs.org/) (推荐 v18 或更高版本)
-   [npm](https://www.npmjs.com/) (通常随 Node.js 一起安装)
-   DeepSeek API 密钥 (从 https://platform.deepseek.com/ 获取)

### 安装步骤

1.  **克隆仓库:**
    ```bash
    git clone https://github.com/venccci/ai-novel-movie.git
    cd AI-Novel-Movie
    ```

2.  **前端设置:**
    进入 `client` 目录并安装依赖：
    ```bash
    cd client
    npm install
    ```

3.  **配置 API 密钥:**
    在 `client/src/App.tsx` 文件中，找到 `API_KEY` 常量，将其替换为您自己的 DeepSeek API 密钥：
    ```typescript
    const API_KEY = "your_deepseek_api_key_here";
    ```

### 运行应用

应用主要运行前端，后端为可选（用于扩展功能）。

1.  **启动前端应用:**
    在 `client` 目录下运行:
    ```bash
    npm run dev
    ```
    前端开发服务器通常会在您的浏览器中打开 `http://localhost:5173`。

2.  **(可选) 启动后端服务:**
    如果需要后端功能，进入 `server` 目录并安装依赖后启动：
    ```bash
    cd server
    npm install
    npm start
    ```
    后端API将在 `http://localhost:3000` 上可用。

## 使用方法

应用将引导您顺序完成四个步骤。每个步骤完成后，下一步骤将自动解锁。

### 第一步：项目与风格设定
定义项目名称，选择视觉风格（日式动漫、3D动画、赛博朋克、黑白素描），设置画幅比例，并输入自定义正向/负向提示词以控制AI生成质量。

### 第二步：小说剧本生成
粘贴小说文本，点击“一键提取剧本”调用 DeepSeek API 分析并生成结构化剧本。支持手动编辑场景地点、时间、角色和描述，可添加或删除场次。

### 第三步：角色设计
从剧本中自动提取角色，或手动添加角色。编辑角色名称、角色类型和特征描述。点击“生成立绘”可生成角色视觉资产（模拟）。

### 第四步：分镜脚本
基于剧本和角色生成分镜列表。每个镜头包含画面描述、镜头类型、运镜方式、音频和标签。可逐个生成镜头预览图。
