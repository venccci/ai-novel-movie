# AI 小说转电影项目

一个基于 AI 的小说转电影脚本和分镜生成工具，支持用户输入小说内容，自动生成剧本、角色设计和分镜脚本。

## 功能特点

1. **项目管理**：新增项目仪表板，支持创建、打开、删除和搜索项目工程
2. **项目设置**：配置电影标题、视觉风格、比例和自定义提示词
3. **小说转剧本**：自动将小说内容拆解为结构化剧本场景
4. **角色设计库**：从剧本中提取角色，包含外貌描述和背景故事
5. **分镜脚本**：生成详细的分镜表，包含画面描述、参数设置和 AI 生图提示词
6. **数据持久化**：支持保存项目进度，每次进入项目可恢复上次保存的状态

## 优化内容

### 1. 剧本结构预览滚动优化
- **问题**：剧本结构预览列表无法滚动，多场景情况下用户只能看到三个场景
- **解决方案**：将容器高度设置为 `min-h-full`，确保 `overflow-y-auto` 能够正常工作，用户可以滚动查看所有场景

### 2. 角色设计库优化
- **问题**：角色设计库中只有背景描述，缺少外貌描述
- **解决方案**：
  - 更新 `Character` 接口，将原来的 `traits` 字段拆分为 `appearance`（外貌描述）和 `background`（角色背景）
  - 修改 AI 角色生成逻辑，请求同时生成外貌描述和角色背景
  - 更新角色设计 UI，显示两个独立的文本框用于输入/编辑外貌描述和角色背景

### 3. 分镜脚本生成优化
- **问题**：生成的分镜表不够完整，小说内容只生成了一部分
- **解决方案**：
  - 更新分镜生成提示词，要求生成全面的分镜列表，覆盖所有场景和重要时刻
  - 将角色信息（外貌和背景）传递给 AI，以便生成更准确的分镜

### 4. 参数设置本地化
- **问题**：分镜脚本中的参数设置选项使用英文
- **解决方案**：将镜头类型和相机运动的选项改为中文，提供更友好的用户体验
  - 镜头类型：特写、中景、全景、近景、远景
  - 相机运动：固定、摇摄、推轨、升降、手持

### 5. 提示词字段添加
- **问题**：缺少可以用于 AI 生图的提示词
- **解决方案**：
  - 在 `Shot` 接口中添加 `prompt` 字段
  - 移除原来的预览字段
  - 修改分镜生成逻辑，为每个镜头生成详细的 AI 生图提示词
  - 在分镜脚本中显示提示词字段，用户可以复制用于其他 AI 生图软件

### 6. 项目管理功能优化
- **新增功能**：
  - 移除顶部“导出项目”按钮，简化界面
  - 新增首页仪表板，作为一级菜单入口
  - 支持项目工程的创建、打开、删除和搜索
  - 点击保存按钮可保存当前项目数据，下次进入时自动加载
- **解决方案**：
  - 扩展后端 projects API 支持完整 CRUD 操作
  - 新增 Dashboard 页面组件，提供项目列表管理
  - 修改导航结构，默认进入首页，原有功能变为二级菜单
  - 实现项目状态保存与加载功能

### 7. 首页项目管理体验优化
- **优化内容**：
  - 确保首页项目名与项目内部名称保持一致
  - 移除首页项目卡片右上角的多余操作按钮（三个点）
  - 优化创建项目逻辑：点击创建按钮后弹出模态框，输入名称后直接进入新项目界面
  - 优化项目卡片交互：点击卡片直接打开项目，移除单独的打开按钮
  - 在项目编辑界面顶部导航栏添加返回首页按钮，提升导航便捷性
  - 首页空状态只保留右上角创建项目按钮，简化界面
  - 强化项目设定流程：必须完成所有基本设定（标题、风格、比例）才能进入下一步
- **解决方案**：
  - 修改后端创建项目逻辑，确保 projectData.title 与项目 title 一致
  - 更新 Dashboard 组件，移除项目卡片上的多余操作按钮
  - 创建 Modal 组件，实现弹出式项目名称输入界面
  - 更新创建项目流程，成功创建后直接打开新项目
  - 为项目卡片添加点击事件，直接触发打开项目功能
  - 更新 TopBar 组件，添加返回首页按钮及相应的导航功能
  - 移除 Dashboard 空状态下的重复创建项目按钮
  - 修改项目设定界面的下一步按钮条件，确保所有必填字段完成
  - 更新步骤状态管理，强化流程管控

### 8. 首页卡片样式与状态优化
- **优化内容**：
  - 移除首页项目卡片上的"未保存"和"进行中"状态标签
  - 改进卡片样式，添加悬停效果
  - 减小卡片尺寸，提升视觉舒适度
- **解决方案**：
  - 更新 Dashboard 组件，移除卡片上的状态显示
  - 调整卡片 CSS 类，将内边距从 `p-6` 减小到 `p-4`
  - 减小标题字体大小，优化空间利用
  - 添加悬停效果（阴影、边框颜色变化），提升交互体验
  - 隐藏删除按钮，仅在悬停时显示，保持界面简洁

### 9. AI 功能异步处理优化
- **优化内容**：
  - 所有 AI 相关功能（剧本生成、角色提取、分镜生成）改为项目内异步处理
  - 用户可以在 AI 处理过程中自由切换项目内的不同菜单
- **解决方案**：
  - 新增 `AITask` 接口，用于全局管理 AI 任务
  - 在 `App.tsx` 中实现 AI 任务的创建、更新和移除功能
  - 修改各步骤组件，使用全局 AI 任务管理替代本地状态
  - 移除按钮禁用状态，允许用户在 AI 处理时继续操作
  - 实现任务状态持久化，确保切换菜单后仍能追踪任务进度

### 10. 分镜脚本 AI 生成优化
- **优化内容**：
  - 解决分镜脚本 AI 生成失败的问题
  - 提升生成结果的可靠性和格式一致性
- **解决方案**：
  - 增强系统提示词，明确要求返回有效的 JSON 格式
  - 添加 JSON 格式验证，确保 AI 返回的数据符合预期结构
  - 处理缺少必要字段的情况，提供默认值
  - 改进错误处理机制，提供更详细的错误信息
  - 添加重试机制，提高生成成功率

## 技术栈

- **前端框架**：React + TypeScript
- **样式框架**：Tailwind CSS
- **构建工具**：Vite
- **AI API**：DeepSeek

## 使用说明

1. **项目设置**：输入电影标题，选择视觉风格和比例，添加自定义提示词
2. **小说转剧本**：粘贴小说内容，点击"一键提取剧本"，AI 将自动生成剧本场景
3. **角色设计**：从剧本中提取角色，编辑角色的外貌描述和背景故事
4. **分镜脚本**：生成分镜表，每个分镜包含画面描述、参数设置和 AI 生图提示词

## 注意事项

- 确保在 `.env` 文件中配置正确的 DeepSeek API 密钥
- 小说内容越详细，生成的剧本和分镜质量越高
- 可以手动编辑剧本、角色和分镜内容，以获得更好的结果
